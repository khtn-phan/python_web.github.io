Practice Exercises

Test Your Knowledge

Apply what you've learned with these hands-on exercises.

Exercise 1: Hello You

Write a program that asks the user for their name and then prints "Hello, [name]!"

Example Output:

What is your name? Alice
Hello, Alice!

Exercise 2: Even or Odd

Write a program that takes a number as input and prints whether it's even or odd.

Example Output:

Enter a number: 7
7 is odd

Exercise 3: Sum of Numbers

Write a function that takes a list of numbers and returns their sum.

Example Output:

>>> sum_list([1, 2, 3, 4, 5])
15
